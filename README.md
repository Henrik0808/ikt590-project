# IKT590 Project - Semi-supervised learning for text classification

Student project for IKT590 Master's Thesis

## Usage (Windows)

```bat
py -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

## Usage (Linux)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
