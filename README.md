# IKT442 Project - Semi-supervised learning for text classification
Student project for IKT442 ICT Seminar 3


## Usage (Windows)
```bat
py -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
